How to play:

- Select the date of the puzzle you'd like to play. There is a new puzzle every day!
- Solve the puzzle by completing each phrase using the given clue. The last letters or word of
  each phrase make up the first letters or word of the next phrase. These "shared"
  characters are grey.
- Press the "Show Letter" button or hit the shift key, followed by a letter, to reveal all instances of
  that letter throughout the entire puzzle. Shown letters are red.

Your score for each puzzle is calculated as follows:
Each Second: -0.1
Show Vowel: -3
Show Consonant: -2

Complete each puzzle with the best score possible!
